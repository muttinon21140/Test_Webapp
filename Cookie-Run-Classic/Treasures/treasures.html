Cookie Run Classic - Treasure

Treasure สำคัญที่ช่วยเสริมทีมใน Cookie Run Classic เลือกของที่เหมาะกับสไตล์เกมของคุณ

กลับหน้าหลัก ไป Maps

Treasure แนะนำ

Cherry Bomb

เพิ่มพลังโจมตีและขนาดระเบิด เหมาะกับ Cookie ที่มีสกิลระเบิด

Golden Bell

เพิ่มคะแนนโบนัสและลดคูลดาวน์สกิล ของสำคัญสำหรับทีมบูสเตอร์

Magic Wand

เพิ่มพลังโจมตีเวทมนตร์และความเร็วเคลื่อนที่ ช่วยให้วิ่งได้รวดเร็วขึ้น
