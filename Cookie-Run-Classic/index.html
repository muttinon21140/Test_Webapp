Cookie

ดู Cookie ทั้งหมด ->

Adventurer Cookie

เหมาะสำหรับการวิ่งทั่วไปและการเก็บคะแนนสูงด้วยสกิลโซนระเบิด

Dark Choco Cookie

บัพพลังโจมตีสูง เหมาะสำหรับบอสและด่านที่เน้นดาเมจ

Matcha Cookie

แรงถึกและฟื้นฟูพลังชีวิต เหมาะสำหรับแนวป้องกันและการอยู่รอดได้ดี

Pet

ดู Pet ทั้งหมด ->

Shiba Cookie

เพิ่มค่าคะแนนและช่วยเก็บเหรียญได้ไวขึ้นในการวิ่ง

Heart Cookie

ช่วยเพิ่มพลังป้องกันและฟื้นฟู HP ระหว่างด่าน

Snow Sugar Cookie

ดีสำหรับระบบสนับสนุนและช่วยสร้างระเบิดน้ำแข็งได้บ่อย

Treasure

ดู Treasure ทั้งหมด ->

Cherry Bomb

เพิ่มพลังโจมตีและขนาดพื้นที่ระเบิด เหมาะสำหรับสกิลระเบิดของ Cookie หลายประเภท

Golden Bell

เพิ่มค่าคะแนนโบนัสและลดระยะเวลาคูลดาวน์ของสกิล

Magic Wand

ช่วยเพิ่มพลังโจมตีเวทมนตร์และความเร็วการเคลื่อนที่
